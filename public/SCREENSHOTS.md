# Add your project screenshots here

The portfolio expects two image files in this `public/` folder. Save the two
screenshots you have with these exact names:

| File name                 | Which screenshot                                  |
| ------------------------- | ------------------------------------------------- |
| `campus-lost-found.png`   | The Campus Lost & Found page (red/green buttons)  |
| `audio-transcriber.png`   | The Audio Transcriber page (Choose File / Transcribe) |

Once the files are here, refresh the page — they appear automatically in the
project cards. Until then, each card shows a dashed "screenshot" placeholder.

- PNG or JPG both work; if you use `.jpg`, update the `image:` paths in
  `src/App.jsx` (the `PROJECTS` array) to match.
- A roughly 16:9 (landscape) crop looks best, but any size works — images are
  cropped to fit from the top.
